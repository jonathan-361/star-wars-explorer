## Star Wars Explorer

Para lograr hacer funcionar el backend y frontend, es necesario el comando `npm run dev`.

### Backend

Solo existe un endpoint para la API creada, para la petición _GET_ será necesario:
**GET:** `http://localhost:3000/api/people/{id}`

Así mismo, se necesita un archivo _.env_, pero está se puede crear en base al archivo _.env.example_.

### Frontend

Para este apartado ya cuenta con todo, solo es necesario realizar las pruebas.
